Map

创建：new Map()

添加：map.set(key, value)

可以连续调用：mmap.set().set().set()
等于mmap.set()
.set()
.set()

获取：map.get(key)

！任意格式数据：无论在哪里

删除：map.delete(key)

大小：map.size => key数量

若不是基本数据类型：

rest.set([1,2], true)
console.log(rest.get([1,2])) // undefined

需提前存变量

另一种创建方法

new Map ([ [key1, value1], [key2, value2], ...]) // 和Object.entries()返回的一样 => 可快速转换

.keys(), .values()适用
