Set

:Unique

去重，合并相同的项。

创建：Set(--)，传数组/对象之类

new Set(['Pasta', 'Pizza', 'Pizza'])

Set [ "Pasta", "Pizza" ]

new Set('hel') ===> Set(3) [ "h", "e", "l" ] 传字符串会分解

set.has() 查询是否存在；true/false

set.size 项数

操作：

增加：set.add(--)

删除：set.delete(--)

retrieve 找回，检索（储存于计算机的信息），挽回；回忆

删除所有：set.clear()

iteration, 可以for .. of 遍历；可以...Set

实际用处

去除数组中重复项

转换数组

用...

[...new Set(Arr)]
