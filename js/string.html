在传入函数时，js会把基本数据类型转换成类似于object的形式

const objectStr = new String('Hello') console.log(objectStr) // String { "Hello" } 0: "H" 1: "e" 2: "l" 3: "l" 4: "o" length: 5 console.log(typeof objectStr) // object

大小写转换

.toLowerCase(), toUpperCase()

首字母大写： const hello = 'hElLo' const helloLower = hello.toLowerCase() const helloCorrect = helloLower[0].toUpperCase() + helloLower.slice(1) console.log(helloCorrect)

.trim()字符串的两端清除空格，返回一个新的字符串
此上下文中的空格是指所有的空白字符（空格、tab、不换行空格等）以及所有行终止符字符（如 LF、CR 等）。

const helllo = ` HellLO ` const hellloTrimmed = helllo.toLowerCase().trim() // helllo .toLowerCase()返回一个字符串，可以继续操作.trim()
