FFmpeg

参数：

$ ffmpeg {1} {2} -i {3} {4} {5}
  1. 全局参数
  2. 输入文件参数
  3. -i 输入文件
  4. 输出文件参数
  5. 输出文件
参数太多的时候，为便于查看，命令可写成多行。 ffmpeg \ [全局参数] \ [输入文件参数] \ -i [输入文件] \ [输出文件参数] \ [输出文件] 最简单的命令：$ ffmpeg -i input.avi output.mp4
全局参数 -c 指定编码器 ffmpeg configure -encoders 查看ffmpeg支持的编解码器 (codec) 列表 - 常用：-c copy // 复制音视频流 - copy 也是-c:a 和 -c:v 的编码器之一 -i: 指定输入文件 - 查看视频文件的元信息，比如编码格式和比特率，可以只使用-i参数。 上面命令会输出很多冗余信息，加上-hide_banner参数，可以只显示元信息。
输出文件参数 编码器 -c:v 指定视频编码器 -c:α 指定音频编码器 -an 去除音频流 -av 去除视频流 码率 -minrate xxxK (bps) -maxrate -bufsize buffer size, 缓冲区, 一般设2000K 可以用-preset ultrafast, superfast, veryfast, faster, fast, medium, slow, slower, veryslow
